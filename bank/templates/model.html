How this model works

This application uses a logistic regression model trained on the UCI Bank Marketing dataset. Logistic regression is a linear model that estimates the probability of a binary outcome (subscribe / not subscribe) by applying the logistic function to a weighted sum of input features.

Feature preprocessing

The original dataset contains categorical fields (e.g., job, marital, education, contact, month, poutcome). During training these were encoded as numeric values and numeric features were optionally scaled. The web form expects inputs in the same encoded format used at training time.
